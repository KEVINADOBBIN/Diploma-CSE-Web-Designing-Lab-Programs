COMPUTER SCIENCE DEPARTMENT

DIGITAL AND COMPUTER FUNDAMENTAL

NUMBER SYSTEM

A NUMBER SYSTEM DEFINES A SET OF VALUES USED TO REPRESENT QUANTITY.

A NUMBER SYSTEM CAN BE CATEGORISED INTO TWO

1. NON POSITIONAL NUMBER SYSTEM
2. POSITIONAL NUMBER SYSTEM

NON POSITIONAL NUMBER SYSTEM

IN ANCIENT TIMES FOR COUNTING PEOPLE USED STONE, PEBBLES OR STICKS WHERE USED TO INDICATE THE VALUES.
SUCH METHOD OF COUNTING IS CALLED NON POSITIONAL NUMBER SYSTEM.

POSITIONAL NUMBER SYSTEM

THE SYMBOL AS WELL AS ITS POSITION DEFINES THE VALUE OF EACH DIGIT IN THE NUMBER.
THEY HAVE BASE OR RADIX.
THE MOST COMMONLY USED NUMBER SYSTEM IS DECIMAL NUMBER SYSTEM (THIS SYSTEM IS BASE 10).
