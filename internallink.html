INTERNET

THE INTERNET IS A GLOBAL SYSTEM OF INTER-CONNECTED COMPUTER NETWORKS THAT USE THE STANDARD INTERNET PROTOCOL SUITE TO SERVE BILLIONS OF USERS WORLDWIDE.

MAINPAGE




WEBSITE

A WEBSITE IS A COLLECTION OF RELATED WEB PAGES, IMAGES, VIDEOS, OR OTHER DIGITAL ASSETS THAT ARE ADDRESSED RELATIVE TO A COMMON UNIFORM RESOURCE LOCATOR (URL).

MAINPAGE




WEBPAGE

A WEBPAGE IS ANY SINGLE PAGE OF INFORMATION ON THE WORLD WIDE WEB. IT IS WRITTEN IN HTML AND MAY INCLUDE TEXT, IMAGES, SCRIPTS, AND OTHER FILES. WEB BROWSERS TRANSLATE HTML FILES INTO VISIBLE CONTENT.
